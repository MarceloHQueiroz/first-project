# First-Project: Simplified_Inventory_Management_System
This is a simple project developed in Java with the goal of practical learning. It simulates a small product management system, focusing on understanding the fundamentals of Object-Oriented Programming (OOP) as well as the basic use of Git and GitHub.
-------

## Project Objective
This project was created as part of my learning journey in Java development.

The main focus is to:
  - Understand the structure of a Java project;
  - Practice Object-Oriented Programming (OOP);
  - Work with classes and objects;
  - Use ArrayList for data storage;
  - Apply the Singleton pattern in a simple way;
  - Learn the basic workflow of version control with Git and GitHub.
-------

## What I Learned from This Project
- How to structure a system in Java from scratch;
- Separation of responsibilities between classes;
- Using lists (ArrayList) to manage in-memory data;
- Basic concepts and application of the Singleton pattern;
- How to version code with Git (commit, push, etc.);
- How to use GitHub to store and organize projects.
-------

## Technologies Used
- Java (OOP)
- ArrayList
- Git
- GitHub
- IDE: Eclipse
-------

## System Features
This system simulates a simple product management process:
  - Product registration (create);
  - Product listing;
  - Product removal;
  - Organized class structure;
  - Data stored in memory (no database);
-------

## Project Structure
src/

├── model/ → Main classes (Product)

├── service/ → Business rules

├── repository/ → In-memory storage (ArrayList)

├── main/ → Main class (Main)  

-------

## How to Run
1. Clone the repository
2. Open the project in your Java IDE
3. Run the Main.java class

# Primeiro-projeto: Sistema Simples de Gerenciamento de Produtos (Java)
Este é um projeto simples desenvolvido em Java com o objetivo de aprendizado prático. Ele simula um pequeno sistema de gerenciamento de produtos, focado em entender os fundamentos da Programação Orientada a Objetos (POO) e também o uso básico de Git e GitHub.
-------

## Objetivo do projeto
Este projeto foi criado como parte do meu processo de aprendizado em desenvolvimento Java.

O foco principal é:
  - Entender a estrutura de um projeto em Java;
  - Praticar Programação Orientada a Objetos (POO);
  - Trabalhar com classes e objetos;
  - Utilizar "ArrayList" para armazenamento de dados;
  - Aplicar o padrão Singleton de forma simples;
  - Aprender o fluxo básico de versionamento com Git e GitHub.
-------

## O que aprendi com este projeto
- Como estruturar um sistema, em Java do zero;
- Separação de responsabilidades em classes;
- Uso de listas (ArrayList) para gerenciar dados em memória;
- Conceitos e aplicações básica de Singleton;
- Como versionar o código  com Git (commit, push, etc.);
- Como usar GitHub para armazenar e organizar projetos.
-------

## Tecnologias utilizadas
- Java(POO)
- ArrayList
- Git
- GitHub
- IDE: Eclipse
-------

## Funcionalidades do sistema
Este sistema simula um gerenciamento simples de produtos:
